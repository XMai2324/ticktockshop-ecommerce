# E-commerce Watch Store

## Description
This is an academic e-commerce web application for selling watches, developed as part of a university project.  
The system supports basic online shopping features and follows a standard system workflow.

## Main Features
- User authentication (login, register)
- Product listing and product detail pages
- Shopping cart and order management
- Basic admin management for products

## Technologies
- Laravel (PHP Framework)
- MySQL
- Visual Studio Code
- XAMPP
